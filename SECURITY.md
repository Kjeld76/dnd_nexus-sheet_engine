# Security Policy

## Supported Versions

Dieses Projekt hat aktuell keine formale LTS-Policy. Bitte nutze möglichst die neueste Version.

## Reporting a Vulnerability

Bitte melde Sicherheitsprobleme **nicht** öffentlich (keine Issues/PRs).

- **Kontakt**: sende eine E-Mail an den Maintainer (Adresse bitte im GitHub-Profil/Repo hinterlegt verwenden).
- **Inhalt**: reproduzierbare Schritte, betroffene Version, Impact, ggf. Proof-of-Concept.

Wir bemühen uns um eine zeitnahe Rückmeldung und Koordination einer Fix-Veröffentlichung.
